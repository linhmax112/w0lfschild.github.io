Dark Boot

Make your Macs boot and login screens pop, on macOS 10.9 and above.

  • Download

  • Github

Dark Boot gives you control over how your mac looks from the lock screen and when starting up. There are tons of options for customization so your mac can look just the way you want it to. Dark Boot is designed to make theming the Boot Screen, Login Screen and Lock Screen easy and robust on macOS 10.9 and up.

There are many ways to configure Dark Boot as you wish. Give it a go and find out!


Installation



- Download the latest release of Dark Boot
- Open downloaded zip file
- Run Dark Boot
- Allow Dark Boot to move itself to the Applications folder
- To theme the Lock Screen you may be required to disable System Integrity Protection, Dark Boot will inform you but cannot automate this
- You may be required to update System Components, Dark Boot will do this for you
- Theme away
