Installation

  • 1. Download the latest release of cDock
  • 2. Open the downloaded zip file
  • 3. Open cDock
  • 4. Allow cDock to move itself to the Applications folder
  • 5. Disable System Integrity Protection
  • 6. cDock will need to install or update it's System Components
  • 7. If you are on 10.13 or below you may re-enable System Integrity Protection
  • 8. Theme away
