Installation

- Download the latest release of cDock
- Open downloaded zip file
- Run cDock
- Allow cDock to move itself to the Applications folder
- You may be required to disable System Integrity Protection, cDock will inform you but cannot automate this
- You may be required to update System Components, cDock will do this for you
- Theme away
